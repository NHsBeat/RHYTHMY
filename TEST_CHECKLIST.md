# RHYTHMY — Тест-чеклист (для проверки руками)

Кнопки указаны для геймпада Xbox. В скобках — клавиша на клавиатуре.
Лицевые кнопки: **A** = нижняя (Z), **B** = правая (X), **X** = левая (A), **Y** = верхняя (S).
Триггеры: **LB/RB** = смена экрана (Q/W), **LT/RT** (E/R). **Menu** = Play/Stop (Enter), **View** = меню (Backspace).

---

## A. Запуск
- [ ] Приложение открывается, окно 640×480, тёмная тема
- [ ] Сверху 6 вкладок: RACK ROLL SONG MIX BRWS SMPL
- [ ] **LB / RB** листают экраны, текст справа сверху (BPM/BAR) не наезжает на вкладки

## B. Сделать звук (базовый трек)
- [ ] **RACK**: D-pad ↑↓ выбирают канал, ←→ меняют громкость
- [ ] Нажми **A** на канале → откроется **ROLL** (Piano Roll)
- [ ] В ROLL: стрелки двигают курсор, **A** ставит ноту — **слышно звук** сразу (превью)
- [ ] Поставь несколько нот, **A** на ноте ещё раз — убирает
- [ ] **Y** удлиняет ноту, **X** укорачивает — нота это сплошной прямоугольник (не нарезка)
- [ ] **B** — назад в RACK
- [ ] **RB** до **SONG**: **A** ставит блок паттерна на такт 1
- [ ] **Menu (Play)** — трек играет, бежит жёлтая полоса, слышны ноты
- [ ] **Menu** ещё раз — стоп

## C. Микшер и эффекты
- [ ] **MIX**: ←→ выбор канала, самый левый — **MASTER**
- [ ] ↑↓ громкость, **A** mute, **B** solo, **LT/RT** панорама
- [ ] На канале нажми **Y** → откроется **редактор эффектов**
- [ ] ↑↓ слот, ←→ поле, **LT/RT** меняют (выбери Reverb, покрути Mix)
- [ ] **A** вкл/выкл слот, **B** закрыть. На воспроизведении слышен эффект

## D. Сэмплы
- [ ] **BRWS** → **Y** переключает в режим **Samples**
- [ ] Видно **дерево папок** (без файлов с `._`), ←→ сворачивает/раскрывает папку
- [ ] **X** — прослушать сэмпл (играет целиком), снизу волновая форма + бежит бегунок
- [ ] **A** — загрузить сэмпл на выбранный канал (помечается LOADED)
- [ ] **SMPL** экран → **A (Z)** анализ → показывает TYPE / ROOT / BPM / CHORD
- [ ] В SMPL ↑↓ выбирают строку, **LT/RT** меняют: START, END, PITCH, BPM SYNC, TARGET BPM
- [ ] На волне видно зелёный (старт) и красный (конец) маркеры
- [ ] Поставь ноты на сэмпл-канал в ROLL — играет твой сэмпл, сверху инфо (если анализировал)

## E. Редактирование (меню View)
- [ ] **Undo** — отменяет последнее действие (поставь ноту → Undo → исчезла)
- [ ] **Redo** — возвращает
- [ ] **Copy** в ROLL копирует ноты канала, **Paste** вставляет (в другой канал/паттерн)
- [ ] **Copy** в SONG копирует паттерн, **Paste** создаёт его копию
- [ ] **Rename** в RACK переименовывает канал, в SONG — паттерн (окно ввода)
- [ ] **Velocity**: в ROLL наведись на ноту, **LT/RT** меняют громкость — яркость ноты меняется

## F. Запись вживую
- [ ] Меню → **Record** = ARMED (сверху загорается REC)
- [ ] **Menu (Play)**, в ROLL двигай курсор по высоте и жми **A** в ритм — ноты записываются на позицию полосы

## G. Темп и грув (меню View)
- [ ] **BPM** ←→ меняет темп
- [ ] **Tap Tempo** → стучи **A** в ритм — BPM подстраивается
- [ ] **Metronome** ON + Play — слышны клики (первая доля выше)
- [ ] **Swing** — на воспроизведении меняется грув
- [ ] **Half Time** ON — играет в 2 раза медленнее и ниже
- [ ] **Tape Stop** → **A** — звук тормозит как плёнка и разгоняется

## H. Проекты (меню View)
- [ ] **Save Project** → окно имени: **LT/RT** курсор, ↑↓ или правый стик меняют букву, **Y** удалить, **A** сохранить
- [ ] **Load Project** → спросит «сохранить?» (Да/Нет/Отмена) → откроется список → выбери проект
- [ ] Загруженный проект восстановился полностью
- [ ] **Export WAV** → создаётся `export.wav` в папке проекта — послушай в плеере
- [ ] Поработай минуту → проверь что появился `projects/_autosave.rhy` (автосейв)

## На что обратить внимание (могут быть шероховатости)
- Тайм-стретч (BPM SYNC = NORMAL) — качество звука
- Точность анализа ноты/аккорда/BPM на разных сэмплах
- Удобство ввода имени проекта
