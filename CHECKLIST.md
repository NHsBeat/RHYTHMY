# RHYTHMY — Чеклист проекта

Портативная DAW для R36S (640×480, управление геймпадом, без курсора).
Стек: C++17, SDL2, miniaudio, CMake. Дата: 2026-06-05.

---

## ✅ Ядро и архитектура
- [x] Сборка CMake + MSVC, окно SDL2 640×480
- [x] Аудио-движок miniaudio (44.1кГц, стерео, float)
- [x] 32 полифонических голоса
- [x] Lock-free очередь нот (аудио-поток не блокирует UI)
- [x] Пошинная обработка: канал → эффекты → громкость/пан → мастер → эффекты мастера → лимитер
- [x] Встроенный bitmap-шрифт (без внешних зависимостей)
- [x] Ввод: клавиатура + геймпад Xbox (+ правый стик)

## ✅ Экраны (6, листание LB/RB)
- [x] **RACK** — Channel Rack (каналы, инструменты)
- [x] **ROLL** — Piano Roll (ноты)
- [x] **SONG** — Playlist (расстановка паттернов)
- [x] **MIX** — Mixer (микшер)
- [x] **BRWS** — Browser (пресеты + сэмплы)
- [x] **SMPL** — Sample (анализ + редактор сэмпла)
- [x] Контекстное меню (View) с настройками и командами

## ✅ Модель данных (как FL Studio)
- [x] Паттерн = контейнер с нотами для всех каналов
- [x] Один паттерн можно ставить много раз (ноты общие)
- [x] Блоки паттернов на таймлайне (float-позиции)
- [x] Уникальные ID нот (наложенные ноты не глушат друг друга)

## ✅ Инструменты
- [x] Синтезатор: 4 осциллятора (Square/Saw/Triangle/Sine) + ADSR
- [x] Сэмплер: воспроизведение .wav, питч по ноте, **стерео**
- [x] 20 встроенных пресетов (5 категорий)
- [x] Mute-группы (choke, как у хэтов)

## ✅ Секвенсор и воркфлоу
- [x] Создание / размещение / перемещение паттернов (правый стик)
- [x] Растягивание паттернов (1–8 тактов)
- [x] Магнит (snap): 1 такт / 1/2 / 1/4 / 1/8
- [x] Маркер старта воспроизведения
- [x] Полоса воспроизведения (playhead) + пульс-индикатор
- [x] Регулировка BPM + **тап-темп** (кнопка A)
- [x] Метроном (вкл/выкл, громкость)
- [x] Свинг / грув

## ✅ Piano Roll
- [x] Ставить/убирать ноты
- [x] Длина ноты (растягивание A/S)
- [x] **Velocity** нот (LT/RT, яркость = громкость)
- [x] Линейка тактов/долей
- [x] Инфо о сэмпле сверху (если анализирован)
- [x] **Запись вживую** (Record + игра в ритм)

## ✅ Микшер и эффекты
- [x] Громкость / панорама / mute / solo на канал
- [x] **Мастер-канал** (слева)
- [x] 3 слота эффектов на канал и на мастер
- [x] Эффекты: **Reverb, Delay, Filter, Distortion, Bitcrush**
- [x] Перф-эффекты: **Tape Stop, Half Time**
- [x] Редактор эффектов (FX editor)

## ✅ Сэмплы
- [x] Браузер папок деревом (как FL), сворачивание/раскрытие
- [x] Предпрослушивание (весь сэмпл) + волновая форма + бегунок
- [x] **Анализ**: BPM, корневая нота, аккорд (14 типов аккордов)
- [x] **Редактор**: начало/конец (обрезка)
- [x] Питч ±24 полутона
- [x] Синхрон BPM: **винил** (питч следует) / **обычный** (тайм-стретч, питч сохраняется)
- [x] Свой целевой BPM
- [x] Очистка macOS-мусора + фильтр в сканере

## ✅ Проекты и файлы
- [x] Сохранение с вводом имени (карусель букв) → папка `projects/`
- [x] Загрузка через браузер проектов
- [x] Вопрос «сохранить?» (Да/Нет/Отмена) перед загрузкой
- [x] **Автосохранение** каждые 60 сек
- [x] **Экспорт в WAV** (16-бит стерео, весь трек)

## ✅ Редактирование
- [x] **Отмена / Повтор** (undo/redo, 30 шагов)
- [x] **Копировать / Вставить** (ноты и паттерны)
- [x] **Переименование** каналов и паттернов
